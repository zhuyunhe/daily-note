# 记录一些移动端常见的CSS问题  

### 点击样式闪动  
```
-webkit-tap-highlight-color : transparent ; 
```   

### 屏蔽用户选择  
```
-webkit-touch-callout: none;
-webkit-user-select: none;
-khtml-user-select: none;
-moz-user-select: none;
-ms-user-select: none;
user-select: none;
```

### 去除移动端IOS输入框的内阴影  
```
-webkit-appearance: none;
```
